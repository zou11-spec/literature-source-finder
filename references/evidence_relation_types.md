# Evidence Relation Types

Use these labels when mapping candidate sources to a user claim. Prefer conservative labels when the abstract or metadata is incomplete.

## direct_support

Use only when the title, abstract, or available metadata clearly reports a finding that matches the user claim. Full-text reading is still required before formal citation.

## theoretical_support

Use when the source provides a concept, theory, framework, or model that can support the user's reasoning without directly testing the same empirical case. Common communication-studies examples include framing, agenda-setting, affordances, mediatization, platformization, affective publics, identity construction, public sphere, narrative, discourse, and city branding.

## empirical_parallel

Use when the source studies a similar platform, audience, context, method, or phenomenon and can be used as a comparable empirical case, but does not directly establish the same claim.

## methodological_relevance

Use when the source is useful mainly because it demonstrates a method, coding scheme, interview approach, discourse analysis strategy, content analysis design, or qualitative interpretation practice.

## contextual_relevance

Use when the source provides background, field context, terminology, historical framing, or adjacent evidence. It may help a literature review but should not be cited as direct support for the claim.

## counterpoint

Use when the source appears to complicate, qualify, contradict, or limit the user's claim. Keep these sources visible because they reduce confirmation bias.

## weak_match

Use when the source only shares broad keywords or metadata is too sparse to judge the relationship. Weak matches should usually appear below stronger candidates or be excluded from a final bibliography.
